Introduction

Dans ce tutoriel, vous apprendrez comment ajouter une redirection simple pour votre site web! Nous aborderons 2 façons de le faire : en passant par le panneau de contrôle de Hostinger et par le CPanel. C’est important d’apprendre à le faire si vous souhaitez que votre site Web ou le chemin d’accès d’un site Web redirige vers un autre site Web ou sur le chemin d’un autre site Web.

Ce dont vous aurez besoin Avant de commencer ce guide, vous aurez besoin des éléments suivants: Accédez au panneau de contrôle de votre compte d’hébergement. Option 1 – Ajout d’une redirection via le panneau de contrôle Hostinger Le processus de configuration d’une redirection dans Hostinger est vraiment rapide et facile. Vous devez connaître l’URL que vous souhaitez rediriger et la nouvelle URL. Étape 1 – Localisation de la section Redirections Pour rediriger votre site Web, vous devez d’abord naviguer vers la section Redirections dans la catégorie Domaines du panneau de contrôle.
